构建 Angular CLI 6 库

1：ng generate library my-library（my-library为库的名字）

2：在projects/src/lib文件夹内创建组件

3：在projects/src文件夹内的 public_api.ts 内 export 创建的组件

4：执行：ng build my-library --prod;创建库

5：在app.module中或其他懒加载模块中引入projects/src/lib内定义的模块（如：MyLibraryModule）

6：在项目里使用时先引入库模块，再用不同的标签引用库组件;

6：如果library里的组件发生变动，需要重新执行build

经验总结

1：在html模板文件的标签里，如果有类似：(click)="isSyncAnimated=!isSyncAnimated" 等简单的事件时， 应用到的变量（如：isSyncAnimated）也要在ts文件内写出来，否则build时会报错

2：Ctrl+k Shift+[：折叠当前函数（局部折叠）

3：Ctrl+k Shift+]：展开当前函数（局部展开）

4：打包：ng build --prod --base-href=http://www.dizmix.com/works/
