构建 Angular CLI 6 库

1：ng generate library my-library
my-library(库的名字)
2：在projects/src/lib文件夹内创建组件
3：在projects/src文件夹内的 public_api.ts引入创建的组件
4：执行：ng build my-library --prod;创建库
5：在app.module中引入projects/src/lib内定义的模块（如：MyLibraryModule）
6：在项目里使用不同的标签引用库组件

ng build my-library --prod npm install angular-library-builder --save-dev
