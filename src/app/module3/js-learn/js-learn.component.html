使用 Renderer2 选取dom元素

1.用id或class名都可以选择标签，但如果标签是组件内的元素，则会剪切标签的内容，从而实现，标签还在，但没了子元素，成为一个空标签
2.如果选取的标签是其他组件的宿主（host）标签，则不会清空标签的内容
3.如果选取的是其他组件内的标签，则会实现第一条的效果
4.querySelector和querySelectorAll属于js原生方法:
this.elem.nativeElement.querySelector('#isH2')选取的范围只是当前组件
document.querySelector('#isH2')选取的范围为整个dom结构
