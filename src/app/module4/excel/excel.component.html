将数据导出为 Excel 表格

方法一，使用 ngx-xlsx

npm install @notadd/ngx-xlsx --save

npm install xlsx --save

npm install file-saver --save

模块文件内引入: NgxXLSXModule

组件内引入：NgxXLSXService

方法二，使用 XLSX

npm install xlsx --save

在组件内声明 XLSX 即可：import * as XLSX from "xlsx";
